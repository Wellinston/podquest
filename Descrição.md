# podquest

O programa atende o pedido proposto para realização do EP1

Possui as seguintes funções:
- Adicionar Podcast
- Adicionar episódio
- Remover
- Mostrar
- Tocar episódio

Não consegui realizar a função shuffle e nem as funções não obrigatórias.

O programa interage com o usuário e tudo é adicionado somente pelo usuário.
